# Security Policy

## Supported Versions

The current versions are supported; NPredicateBuilder follows Microsoft's policy.

| Version | Dotnet | Supported          |
| ------- | ------ | ------------------ |
| 0.9.7   | 8/9/10 | :white_check_mark: |
| 0.9.6   | 7/8/9  | :x:                |
| < 0.9.5 | < 7    | :x:                |

## Reporting a Vulnerability

NPredicateBuilder will be normally updated on each dotnet major update.

The exception being if a sub-package has a vulnerability.

Please create an Issue to make a report.
